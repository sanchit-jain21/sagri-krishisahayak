Design a complete modern, professional, and user-friendly UI/UX for an advanced smart agriculture platform called “SAGRI – Krishi Shayak”, an India-focused intelligent farming assistant.

The platform should be designed as a real-world scalable product solving major problems faced by Indian farmers using AI and data-driven insights.

CORE FEATURES (AI-based, future model training, no API dependency):
- Crop Disease Detection using image upload (CNN-based UI flow)
- Crop Failure Risk Prediction using soil, weather, and historical data
- Crop Price Forecasting with future trend graphs
- Smart Crop & Treatment Recommendations

MULTI-LANGUAGE + VOICE:
- Full multilingual support (Hindi, English, Punjabi, etc.)
- Voice assistant (mic button on every page)
- Voice input + voice response system
- AI chatbot for farmer queries

USER ROLES:
1. Farmer Panel (Very simple, low-tech friendly UI)
- Upload crop images
- Enter crop/environment details
- View predictions (disease, risk, price)
- Voice assistant + chatbot
- History of predictions
- Alerts (weather, disease outbreak, price drop)
- Offline mode indicator

2. Admin Panel (Advanced analytics dashboard)
- Total users and active farmers
- Village-wise usage stats
- India map with real-time usage visualization
- Crop trends and disease analytics
- Model performance monitoring
- User management system
- Data insights dashboard

AUTHENTICATION:
- Popup-based login
- Mobile number login with OTP
- Role selection (Farmer/Admin)
- After login show feature selection popup

SCREENS TO DESIGN:
- Landing page (hero + CTA + features)
- Login popup (OTP based)
- Feature popup after login
- Farmer dashboard (card-based UI)
- Image upload screen
- Data input form screen
- Prediction results screen (graphs + suggestions)
- Admin dashboard (charts + maps + tables)

UI DESIGN REQUIREMENTS:
- Clean, modern, minimal UI
- Green agriculture gradient theme
- Soft shadows, rounded cards
- Mobile-first responsive design
- Simple UX for farmers with low technical knowledge
- Use icons, illustrations, and clear typography

SYSTEM ARCHITECTURE (VISUAL):
User → Frontend → Backend → AI Models → Database → Results

BACKEND STRUCTURE:
- Flask / FastAPI backend
- Machine learning models:
  - CNN for disease detection
  - Classification for risk prediction

DATABASE STRUCTURE:
- Users (Farmer/Admin)
- Crop Data
