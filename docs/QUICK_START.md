## 📥 **What You Need to Install**

### **1. Git**
**Download:** https://git-scm.com/
**Check:** `git --version`

### **2. Node.js (v20+)**
**Download:** https://nodejs.org/ (LTS version)
**Check:** `node --version` and `npm --version`

### **3. VS Code**
**Download:** https://code.visualstudio.com/

**✅ All installed? Continue!**

---

## 🎯 **Run Locally - 5 Commands**

### **Open Terminal and run:**

```bash
# 1. Navigate to Projects folder
cd C:\Projects
# Mac/Linux: cd ~/Projects

# 2. Clone your repository (REPLACE YOUR-USERNAME!)
git clone https://github.com/YOUR-USERNAME/sagri-platform.git

# 3. Enter project folder
cd sagri-platform

# 4. Install dependencies (wait 2-5 minutes)
npm install

# 5. Start development server
npm run dev
```

**✅ Open browser: http://localhost:5173/**

**Your SAGRI app is running! 🎉**

## 🚀 **Deploy to Vercel - 3 Steps**

### **Step 1: Sign Up**
Go to: https://vercel.com/signup
Click "Continue with GitHub"

### **Step 2: Import Project**
1. Click "Add New" → "Project"
2. Select `sagri-platform`
3. Click "Deploy"

### **Step 3: Done! 🎊**
**Your app is live at:**
```
https://sagri-platform.vercel.app
```

**Auto-deploys on every GitHub push!** ✅

## 🆘 **Common Issues**

### **"npm: command not found"**
➜ Install Node.js: https://nodejs.org/

### **"Port already in use"**
➜ Close other terminals or use: `npm run dev -- --port 3000`

### **Build errors**
```bash
# Fix 99% of issues:
rm -rf node_modules package-lock.json
npm install
```

### **"Cannot find module"**
➜ Run: `npm install`
